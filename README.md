# Projet-POO
Application permettant la gestion d'une base de données d'un Centre d'Adoptions d'Animaux.

Ce projet à été réalisé par :
- BITANE Malak
- EL MAGANA Fatima-Zahra
- EL HAMDI Khawla

Et encadré par : Mr. Mustapha HAIN

Cette application permet de :  
- Ajouter, modifier et supprimer des informations d'une table donnée (Adopteurs, Animaux et Adoptions)  .
- Visualiser la base de données dans une interface claire et intuitive.

Technologies utilisées :
- Python
- SQLite  (pour créer la base de donnée)
- Tkinter (pour créer l'interface)
- PIL (pour manipuler les images utilisées dans l'interface)
